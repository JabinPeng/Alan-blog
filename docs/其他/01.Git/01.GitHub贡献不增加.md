git 为不同的项目设置不同的用户名和邮箱 - qq_2300688967的博客 - CSDN博客：

	1，找到项目所在目录下的 .git/文件夹，进入.git/文件夹，然后执行如下命令分别设置用户名和邮箱：
	git config user.name "ordinaryA"
	git config user.email "353458582@qq.com"
	然后执行命令查看config文件：cat config
